## environment

- Language: JavaScript
- Package Manager: npm 10.2.4
- Library: React.js

### local

### 프로젝트 가져오기

```
git clone https://github.com/LingoConnect/LingoConnect.git
cd LingoConnect
cd frontend
```

### forntend 디렉토리 최상단에 `.env` 파일 생성

```
NODE_ENV : 개발 환경을 지정합니다.
BASE_URL : API 서버의 기본 URL을 설정합니다
```

### 프로젝트 실행

```
npm install
npm start
```
